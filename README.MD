## Setup
    To run this project, install it locally using npm:

    ```
    $ git clone https://github.com/WTL04/Mind-Controlled.git MindControlled
    $ cd MindControlled
    $ pip install -r requirements.txt
    ```

    Run the program using:
    ```
    $ python gui.py
    ```
